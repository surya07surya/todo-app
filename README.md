# React Todo App

A simple and user-friendly Todo List application built with React to manage daily tasks efficiently.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and simple UI

## Technologies Used
- React
- JavaScript
- CSS

## How to Run Locally
1. Clone this repository  
2. Navigate to the project folder  
3. Run `npm install`  
4. Run `npm start`  

## Live Demo
URL 'https://surya07surya.github.io/todo-app/'

## Project Structure
- Components-based architecture
- Reusable React components
- State management using React hooks
